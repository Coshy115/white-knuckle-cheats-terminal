# White Knuckle Cheats Terminal
A TUI that executes commands in White Knuckle through the in-game developer console.

## Usage
1) Download executable file.
2) Open executable file.
3) Start White Knuckle game.

**Note: You have to be in a scene where the console is accessible, like in a level.**

## Things to Keep in Mind
- The console is opened with shift+backtick, which is the default overlay binding for Discord, so that will need to be disabled prior to running this program.
- The game doesn't load the abyss levels right away like all the other levels. The abyss levels are loaded upon entering the Habitation Ending, while simultaneously offloading the previous levels. Because of this, you can no longer teleport to previous levels once you've entered Habitation Ending.
